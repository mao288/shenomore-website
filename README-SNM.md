She No More — Official Website

Official website for **She No More**, a symphonic metal band from Mexico.

🎸 About

She No More is a Mexican symphonic metal band. This website serves as the band's main online presence, featuring their music, story, and visual identity inspired by the *Voice For The Voiceless* album.

🌐 Features

- Hero section with band branding
- Discography section
- Band member profiles
- Links to social media and streaming platforms
- Fully responsive design
